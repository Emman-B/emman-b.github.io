LinkedIn Job Application Tracker Extension

GitHub

This is a Chrome extension that keeps track of jobs that have been applied to via LinkedIn. When applying to a job via LinkedIn, it parses the job's page to keep track of the name of the company, the job title, the method in which you have applied, the URL of the job listing, and the date in which you have applied. You can then copy a CSV-formatted version of your tracked applications to be pasted into any software for tracking, such as a spreadsheet.

This extension was developed primarily using only JavaScript, HTML, and CSS. I was the sole developer of this extension and I had two goals for this project: learning Chrome extension development, and speeding up the job application process for myself.

OTSecret

Try it out GitHub

This is a secret-sharing web application using React, Express, and Node.js. Users can share temporary secret messages that can only be revealed once by the recipient before the server deletes said secret. Secrets have a set expiration time that, if expired, will not allow the recipient to reveal the secret.

All passwords and secret messages are encrypted before it is stored in a PostgreSQL database.

I was the sole developer of this web application, and I created this to learn more about deploying web apps through 3rd-party services (e.g., GitHub Pages and Heroku).
